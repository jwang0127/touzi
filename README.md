# 证衡研究台

一个面向 A 股中长期研究的可复现证据与独立审计基础项目。当前版本包含：

- `MockAdapter` 与合成数据校验器；
- 五道独立审计闸门，裁决仅为 `PASS / REVIEW / BLOCK`；
- 腾讯财经只读行情适配器（自动识别沪深前缀，排除北交所与科创板）；
- 3000 元本金的持仓、盈亏与每日趋势检查；
- 响应式网页研究台与对抗测试。

`PASS` 只表示研究证据流程通过审计，不代表建议买入。项目不承诺收益、不连接券商、不自动下单。

## 运行网页

```powershell
npm install
npm run dev
```

访问 `http://localhost:3000`。生产构建与网页测试：

```powershell
npm test
```

## 运行研究内核测试

```powershell
python -m pip install -e . --no-deps
python -m unittest discover -s tests -p "test_*.py" -v
```

## 腾讯行情说明

用户提供的 `qt.qtimg.cn` 当前无法解析，因此实现使用实测可用的 `https://qt.gtimg.cn/q=`。该接口不是交易所法定披露源，默认只能作为行情参考；关键事实仍需与交易所或法定披露来源交叉验证。

## 持仓检查边界

录入股票代码、成本价、股数和买入日期后，网页将持仓保存在当前设备，并在每天打开时自动读取最新日线。固定规则包括：+50%目标线、-8%止损线、30天期限和20日趋势。50%是高风险目标线，不是预期收益或保证。
