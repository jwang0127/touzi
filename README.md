# 证衡研究台

一个面向 3000 元本金的 A 股自动筛选与每日持仓复核项目。当前版本包含：

- `MockAdapter` 与合成数据校验器；
- 五道独立审计闸门，裁决仅为 `PASS / REVIEW / BLOCK`；
- 腾讯财经只读行情适配器（自动识别沪深前缀，排除北交所与科创板）；
- 沪深主板自动初筛、四维透明评分与数据源故障降级候选池；
- 固定窗口三浪实验模型及20交易日滚动回测（当前仅作辅助，不计入总推荐分）；
- 3000 元本金的持仓、盈亏与每日趋势检查；
- 响应式网页研究台与对抗测试。

`PASS` 只表示研究证据流程通过审计，不代表建议买入。项目不承诺收益、不连接券商、不自动下单。

## 运行网页

```powershell
npm install
npm run dev
```

访问 `http://localhost:3000`。生产构建与网页测试：

```powershell
npm test
```

## 运行研究内核测试

```powershell
python -m pip install -e . --no-deps
python -m unittest discover -s tests -p "test_*.py" -v
```

## 腾讯行情说明

用户提供的 `qt.qtimg.cn` 当前无法解析，因此实现使用实测可用的 `https://qt.gtimg.cn/q=`。该接口不是交易所法定披露源，默认只能作为行情参考；关键事实仍需与交易所或法定披露来源交叉验证。

## 自动推荐与持仓检查边界

网页打开后先自动筛选沪深主板，排除北交所、科创板、ST、退市整理及一手超过 3000 元的股票。评分拆成趋势、估值、流动性和风险四项；若全市场初筛源失效，则切换到已审查候选池并重新读取腾讯最新日线，页面会明确显示当前模式。

采用候选后，模拟持仓保存在当前设备，并在每天打开时自动读取最新日线。固定规则包括：+50%目标线、-8%止损线、30天期限和20日趋势。50%是高风险目标线，不是预期收益或保证。系统不读取财报全文和突发公告，采用前仍需核对交易所披露。

三浪模型将第一段上涨、25%–78%回撤不破起点、接近或突破第一浪高点和量能确认转换为固定规则。12只候选、28个滚动样本的探索回测中，加入三浪后20日平均收益略升，但正收益率和10%命中率下降，因此当前权重为0，只显示辅助信号。
